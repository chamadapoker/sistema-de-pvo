# Fontes de Dados do PVO Modern

Este documento lista as fontes confiáveis para enriquecimento do banco de dados de equipamentos, imagens e informações históricas.

## 1. Armas Nacionais
**URL:** [https://www.armasnacionais.com/](https://www.armasnacionais.com/)

**Descrição:**
Site dedicado à história e dados técnicos dos equipamentos utilizados pelas Forças Armadas Brasileiras.

**Conteúdo Relevante:**
- **Aeronaves:** Fichas completas de caças (F-5, Mirage, AMX, Gripen), aviões de treinamento e transporte.
- **Blindados:** Dados sobre tanques (Leopard, M60) e blindados sobre rodas (Guarani, Cascavel).
- **Artilharia:** Informações sobre sistemas como Astros II e obuseiros.
- **Detalhes:** Inclui histórico de desenvolvimento, especificações técnicas (velocidade, armamento, motorização) e contexto operacional no Brasil.

**Ação Recomendada:**
Utilizar como fonte primária para validar e enriquecer as descrições e estatísticas dos equipamentos brasileiros no banco de dados.

## 2. War Thunder Wiki
**URL:** [https://wiki.warthunder.com/](https://wiki.warthunder.com/)

**Descrição:**
Wiki oficial do jogo War Thunder, contendo dados técnicos detalhados de veículos militares modernos e antigos.

**Conteúdo Relevante:**
- **Imagens:** Renders de alta qualidade e "Garage Shots" com fundo transparente ou neutro.
- **Performance:** Dados de performance (embora gamificados, muitas vezes baseados em documentos reais).
- **Modelos 3D:** Referência visual para identificar variantes.
